저장 후 GitHub Pages 등에 올리면 끝. 별도 빌드·서버 불필요한 단일 파일입니다.
⚙️ 자동화하려면 — 관세청 Open API
공공데이터포털 관세청 수출입실적/품목별 무역통계 API를 FastAPI(예: Railway)에서 호출해 위 DATA와 동일한 JSON 스키마로 가공 → 정적 HTML이 fetch()로 읽도록 하면, 발표일마다 손대지 않고 갱신됩니다. 브라우저 직접 호출은 CORS·인증키 때문에 막히므로 백엔드 경유가 정석입니다.
